	e 要是 list

Operators:
	減 乘 除 MOD 加負號 -> 只有 int 能做這些

	always return a Boolean value (False being 0 and True being 1)
	對 int bool -> 數學操作
	對 list string -> 字典序比較

Build-in Functions:
	len -> 只能用在 list string
